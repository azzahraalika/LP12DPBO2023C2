## ALASAN 
Pada Tugas Masa Depan DPBO framework yang dipakai adalah MVVM (Model-View-ViewModel). 

Karena struktur MVVM itu dari model, view, dan viewModel terpisah jadi mudah dipahami dan tidak ada keterkaitan antara view dengan model sehingga ketika ingin mengubah tampilan tidak akan mengganggu package model. Jadi berasa lebih terstruktur aja ngodingnya karena masing-masing package udah ada tugasnya masing-masing.
